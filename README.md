# Quiz-Portel
🔐 Secure Quiz Portal with Proctoring System
🚀 Overview

Quiz-Portel is a web-based online examination system that allows students to take quizzes securely while enabling instructors to monitor performance.

It includes a low-level proctoring system that detects and prevents cheating using browser-based controls.

🎯 Objectives
Conduct secure online exams
Prevent cheating using browser monitoring
Provide performance analytics
Simulate a real exam environment

✨ Features

👨‍🎓 Student Module
User Login / Signup
View available quizzes
Search & filter quizzes
Attempt quiz with timer
View past performance

👨‍🏫 Instructor Module
Create new quizzes
View student results
Monitor exam logs
Analyze performance

🛡️ Proctoring System
❌ Tab switching detection
❌ Copy / Paste blocking
❌ Right-click disabled
❌ Keyboard shortcuts blocked (Ctrl+C, Ctrl+V, F12)
⚠️ Warning system (Max 3 warnings)
🚨 Auto exam submission after violations

🤖 AI Feature
Auto-generate quiz based on user input topic

🛠️ Technologies Used
HTML → Structure
CSS → Styling & UI
JavaScript → Logic & functionality
LocalStorage → Data storage
